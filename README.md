# Usage

1. Feed [`1_generation_script.md`](./1_generation_script.md) with project description to make PRD.
2. Combine [`2_default_prompt.md`](./2_default_prompt.md) with generated PRD to create project.
3. Optionally feed [`3_audit_script.md`](./3_audit_script.md) to existing project to make audit PRD.
4. Optionally run [`4_screenshot_script.md`](./4_screenshot_script.md) on existing projects with CLI tooling to generate terminal screenshots.
