# Worldwide iPhone-First Personal App Loader

## Assumptions

- MVP uses Apple-supported ad hoc distribution to registered devices, not jailbreaks, private APIs, or security bypasses.
- End users bring their own paid Apple Developer Program account and import an App Store Connect API key.
- There is no local macOS companion app; build and signing work runs on remote macOS workers with Xcode 16.2 or newer.
- Stack: Swift 6, SwiftUI iOS app (`iOS 18.0+`), Vapor 5 API, Fluent + PostgreSQL, Redis queue, S3-compatible object storage.
- MVP accepts `.zip` uploads containing one iOS app Xcode project or workspace that builds with Swift Package Manager dependencies only; CocoaPods and multi-app workspaces are out of scope for MVP.
