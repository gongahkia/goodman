TIMEZONE CONVERTER EXTENSION - Product Requirements Document (Todo Format)
================================================================================
Project: Browser extension for instant, offline, cursor-highlight timezone conversion.
Philosophy: Privacy-first (no AI/cloud), Omnivorous Parsing (handle any format), Frictionless (highlight -> result).
Tech Stack: TypeScript, Vite, Preact (Options/UI), Chrono-node (Parsing), Luxon (Time Logic), Shadow DOM.
Constraint: NO AI. Pure algorithmic/regex parsing.
================================================================================

PROJECT SETUP & INFRASTRUCTURE
Initialize Git repository with standard .gitignore for Node/WebExtension
Setup TypeScript project with Vite for bundling (Multi-page entry: popup, options, content)
Configure ESLint and Prettier with strict type checking rules
Setup Vitest for unit testing parsing logic (crucial for "all formats" requirement)
Create manifest.json (Manifest V3) with minimal permissions (activeTab or scripting)
Configure "storage" permission for saving user preferences
Configure "scripting" permission for injecting logic into host pages
Setup GitHub Actions for build and lint checks on PR
Create release script to zip extension for Chrome Web Store and Firefox Add-ons
Initialize standard README with architectural overview and contribution guide

CORE PARSING ENGINE (The "Omnivorous" Parser)
Research and document edge-case formats (Military, ISO8601, RFC2822, Slack-style relative, etc.)
Install and configure `chrono-node` for natural language parsing (e.g., "tomorrow at 5pm")
Implement custom strict-regex pre-parser for military time formats (often missed by NLP libs)
Implement regex parser for epoch timestamps (seconds and milliseconds)
Implement parser for "Unix Timestamp" labeled strings
Implement heuristic for ambiguous date formats (DD/MM vs MM/DD) based on browser locale
Create a "Format Normalizer" to clean selected text (remove trailing punctuation, excess whitespace)
Implement support for timezone abbreviations (EST, PST, CET) mapping to IANA zones
Create a comprehensive test suite of 50+ date/time string examples to verify parser accuracy
Implement support for relative time parsing ("in 2 hours", "30 mins ago")
Optimize parser performance to run under 10ms to prevent UI jank
Implement caching mechanism for recently parsed strings to avoid re-parsing identical selections

TIMEZONE LOGIC & CONVERSION
Install `luxon` for robust IANA timezone handling and math
Create a helper to detect User's System Timezone reliably
Implement logic to resolve "Ambiguous Time" (e.g., during DST fall-back)
Create a "Target Timezone" manager (allows converting TO specific zones, not just local)
Implement conversion logic: Input Date -> UTC -> Target Zone(s)
Handle "Date Rollover" detection (alert user if converted time is Next Day or Previous Day)
Implement logic to display relative difference ("(Tomorrow)", "(-5 hours)")
Create map of common deprecated timezone names to modern IANA equivalents
Implement "Home Base" logic: Allow user to set a "Home" timezone distinct from current system time

CONTENT SCRIPT & INTERACTION (The "Converts It Box")
Create `selection` event listener with debouncing (200ms) to detect text highlighting
Implement logic to discard selections that are too long (>100 chars) or empty
Implement logic to ignore selections inside editable fields (inputs, textareas) to prevent obstruction
Create a robust coordinate calculator to position the "Converts It Box" (Tooltip) near the cursor
Implement "Smart Positioning" (flip tooltip above/below/left/right to stay in viewport)
Create the Tooltip container using Shadow DOM to isolate styles from the host page
Implement "Fade In" and "Fade Out" animations for the tooltip
Add a small "Pin" button to the tooltip to keep it open while scrolling
Implement "Click Outside" logic to dismiss the tooltip immediately
Handle `z-index` wars: Ensure tooltip attempts to sit above maximum page z-index
Implement "Quick Copy" behavior: Click the result to copy to clipboard
Add visual feedback (checkmark) upon copying
